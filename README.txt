Dashboard v15 - Google Sheets persistente

Correções:
- Ao clicar em Atualizar direto da planilha, a base é salva no navegador (localStorage).
- Ao apertar F5, o dashboard tenta atualizar automaticamente pelo Google Planilhas.
- Se a internet falhar, mantém a última base salva no navegador.
- Continua removendo duplicidades pelo REDS.

Uso:
1. Abra index.html.
2. Aguarde a atualização automática ou clique em Atualizar direto da planilha.
3. Após F5, os dados permanecem.
